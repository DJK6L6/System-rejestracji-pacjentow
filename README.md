# ***System rejestracji pacjentów na badania***  
***Data wykonania: 07.05.2025***

## ***Opis projektu***  
Projekt przedstawia prosty system rejestracji pacjentów na badania medyczne w formie aplikacji webowej.  
Strona umożliwia:  
- wprowadzenie danych pacjenta poprzez formularz,  
- zapisanie wizyty w bazie danych relacyjnej (H2),  
- wyświetlenie aktualnego stanu bazy,  
- przegląd pacjentów zapisanych na wybrany termin.
